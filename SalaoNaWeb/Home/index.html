A Empresa

O Salão na Web é um site que visa comodidade para usuários que estão sem tempo para ligar no salão ou passar no mesmo para agendar um serviço. Quantas vezes já esquecemos de ligar para agendar e deixar para ultima hora? Estamos aqui para resolver seu problema.

Nosso sistema permite para os usuário o agendamento de amplos serviços de beleza no salão desejado sem ter que sair de casa ou fazer uma ligação, acessando apenas o nosso site e escolhendo o salão e o serviço, muito bom, nao acha?
